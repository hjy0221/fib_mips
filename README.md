# fibonacci mips asm
📌 개요

이 프로그램은 MIPS 어셈블리로 구현된 피보나치 수(Fibonacci number) 계산 프로그램입니다.
사용자로부터 정수를 입력받아 재귀 호출(recursion)을 이용하여 해당 번호의 피보나치 값을 계산하고 출력합니다.

📁 파일 설명
main 함수

“Give a number:” 문구 출력

정수 입력받기

입력된 값을 fib 함수에 전달하여 결과 계산

반환된 피보나치 수를 출력

fib 함수

재귀적으로 피보나치 값을 계산

종료 조건:

fib(0) = 0

fib(1) = 1






5


출력:

5
